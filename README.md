ShooterSubX
===========

## Features:

- 利用射手网 API 下载字幕。
- 可设置下载字幕优先级。
- 支持多视频字幕一起下载。
- 可拖拽文件夹到程序主窗口。
- 不会下载有延迟的字幕。
- 下载的字幕在所选择的视频目录下。

## Usage:

- 把视频拖进程序窗口内（可以选择多个视频文件或文件夹）。
- 点击 `Download all` 下载字幕。
