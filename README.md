ShooterSubX
===========
## Features:

- 利用射手网 API 下载字幕。
- 优先选择 ASS 扩展名的字幕，如果没有则下载 SRT 扩展名的字幕。
- 支持多视频字幕一起下载。
- 不会下载有延迟的字幕。
- 下载的字幕在所选择的视频目录下。

## Usage:

- 把视频拖进程序窗口内（可以选择多个视频文件或文件夹）。
- 点击 `Download all` 下载字幕。
